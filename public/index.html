API exposant les données Affelnet Paris

Ce site héberge les données spécifiques à Affelnet Paris (seuils d'admission, statistiques, etc.). Vous trouverez ci-dessous les liens directs vers les fichiers de données au format JSON. Pour une consultation plus aisée, vous pouvez transformer ces données au format tabulaire (CSV) à l'aide d'un outil en ligne comme celui-ci.

Points d'accès de l'API

  • api/colleges.json Informations et statistiques sur les collèges (IPS...).
  • api/lycees.json Informations sur les lycées (seuils d'admission...).
  • api/notes.json Barèmes de conversion des notes brutes en notes harmonisées pour le calcul du score.
  • api/secteur1_2025.json Sectorisation des lycées par collège pour 2025 et bonus associés.
  • api/seuils_recents.json Derniers seuils d'admission connus pour les différents lycées.
  • api/stats_recentes.json Statistiques académiques (moyenne, écart-type) pour l'année 2025.
  • api/statistiques.json Historique des statistiques académiques des années précédentes.
  • doc/prompt.txt Prompt pour IA Générative (testé avec Gemini 2.5 Pro) pour calculer vos admissions à partir de bulletins trimestriels (PDF ou Images).
